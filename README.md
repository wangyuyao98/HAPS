# History-Aware Conformal Prediction Sets for Censored Time-to-Event Outcomes

This repository contains code for reproducing the simulation and data analysis in "History-Aware Conformal Prediction Sets for Censored Time-to-Event Outcomes" by Yuyao Wang, Alexander W. Levis, Shu Yang, Larry Han (2026),  [arXiv:2605.06581](https://arxiv.org/abs/2605.06581)

This note documents the code needed to reproduce the main-paper simulation and
data application results from scratch. The repository does not need precomputed
`.rds` result files; all result files are generated by the commands below.

Some internal filenames still use the historical name `DynaCoPS`. In the paper
and plot legends, these methods are labeled as HAPS, HAPS-DR, and HAPS-A.

## Software

The simulations were run with R and CPU-only Python/PyTorch for the DDH
prediction model.

## Simulation: Small Smoke Test

Run these commands from the repository root. They use `R = 5`, `n = 500`, and
`n_test = 100`; the numerical results are only for checking that the pipeline
runs end to end. 

```bash

Rscript main_derive_HAPS_DR_linWB1.R cox_cox_multiclass_loc0 5 500 100
Rscript main_simu_static_IPCW0_linWB1.R cox_cox_multiclass_loc0 5 500 100
Rscript main_simu_static_variants_linWB1.R cox_cox_multiclass_loc0 IPCW_trunc LM_IPCW 5 500 100

RETICULATE_PYTHON_ENV=r-ddh Rscript main_simu_dynamic_DDH_tuning.R mixC2_DGM1 wide64_epochs50 5 500 100 rsf xgb_cox xgb_multiclass 1
```

## Simulation: Full Main-Paper Run

These commands reproduce the main-paper simulations with `R = 200`,
`n = 1000`, and `n_test = 500`.

### DGM1

DGM1 uses Cox prediction, Cox censoring model, Cox event-survival nuisance
models, and XGB multiclass augmentation nuisance models.

```bash
Rscript main_simu_dynamic_linWB1_cox_multiclass_loc0.R 200 1000 500
Rscript main_derive_HAPS_DR_linWB1.R cox_cox_multiclass_loc0 200 1000 500
Rscript main_simu_static_IPCW0_linWB1.R cox_cox_multiclass_loc0 200 1000 500
Rscript main_simu_static_variants_linWB1.R cox_cox_multiclass_loc0 IPCW_trunc LM_IPCW 200 1000 500
```

### DGM2

DGM2 uses the tuned DDH prediction model with RSF censoring model, XGB-Cox
event-survival nuisance models, and XGB multiclass augmentation nuisance models.
The static comparison methods use RSF prediction and a global baseline-only RSF
censoring model.

```bash
RETICULATE_PYTHON_ENV=r-ddh Rscript main_simu_dynamic_DDH_tuning.R mixC2_DGM1 wide64_epochs50 200 1000 500 rsf xgb_cox xgb_multiclass 10
Rscript main_simu_static_IPCW0_globalG_mixC2_DGM1.R mixC2_DGM1 rsf_rsf_reg_loc0 200 1000 500
Rscript main_simu_static_variants_mixC2_DGM1.R mixC2_DGM1 rsf_rsf_reg_loc0 LM_IPCW 200 1000 500
```

## File Organization

All R and Python source files needed by the main-paper simulation scripts are
kept in `src/`. The top-level `main_*.R` files are executable scripts that call
these helper files and write outputs under `results/`.

## Data Applications

The data application scripts use the `pbcseq` and `colon` datasets distributed
with the `survival` R package, so no external data files are required. The
application analyses report IPCW-estimated coverage because the true event times
are not observed.

The main-paper application plots use DDH prediction for HAPS and HAPS-DR with
time features, the tuned `wide64_epochs50` DDH preset, and a Cox censoring model.
The baseline methods use Cox prediction and a Cox censoring model with baseline
covariates and time-zero covariate values only.

### Application Smoke Test

These commands use `R = 5` random splits and are intended only to check that the
pipeline runs.

```bash
RETICULATE_PYTHON_ENV=r-ddh Rscript main_app_DDH.R all 5 wide64_epochs50 time cox 1
Rscript main_app_generic.R all 0.2 5 cox cox static TRUE
Rscript main_plot_app_mainpaper_covlen.R 0.2 5 wide64_epochs50 time cox
Rscript main_table_app_length_ratios.R 0.2 5 wide64_epochs50 time cox
```

### Full Main-Paper Application Run

These commands reproduce the main-paper application figures and length-ratio
table with `R = 200` random splits.

```bash
RETICULATE_PYTHON_ENV=r-ddh Rscript main_app_DDH.R all 200 wide64_epochs50 time cox 10
Rscript main_app_generic.R all 0.2 200 cox cox static TRUE
Rscript main_plot_app_mainpaper_covlen.R 0.2 200 wide64_epochs50 time cox
Rscript main_table_app_length_ratios.R 0.2 200 wide64_epochs50 time cox
```

Application figures are saved under:

```text
results/app_pbcseq/plots/
results/app_colon/plots/
```

The application length-ratio table is saved under:

```text
results/mainpaper_tables/
```

HAPS-A is not included in the main-paper application workflow. The AIPCW
application analyses required additional approximation of irregular covariate
histories and showed unstable estimating curves in diagnostics, so those
exploratory scripts are intentionally excluded from this compact reproduction
folder.
